# Project Magpie 

**Project Magpie** (formerly Spot It World) is an AI-powered scavenger hunt generator for families. 

## The Concept
Turn any location into a game.
1. **Describe your location:** "Sheffield Park, East Sussex"
2. **AI Generates the Hunt:** The app identifies flora, fauna, and objects specific to that spot.
3. **Play:** A Bingo-style grid where users tick off items.

## Architecture
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** MVVM + Clean Architecture
- **Data:** Room Database (Local) + AI Service (Remote)

## Getting Started
1. Clone the repo.
2. Open in Android Studio (Ladybug or newer).
3. Run on Emulator or Device.
